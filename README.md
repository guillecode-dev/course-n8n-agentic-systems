## 🚀 Cómo usar este repositorio

### 1. Clona el repositorio

```bash
git clone https://github.com/guillecode-dev/course-n8n-agentic-systems.git
```

### 2. Navega al módulo que estás estudiando

```bash
cd course-n8n-agentic-systems/modulo-01-introduccion/ejemplos/
```

### 3. Importa el flujo en N8N

1. Abre tu instancia de **N8N** (local o en la nube)
2. En el editor de flujos, haz clic en el menú **≡** (arriba a la izquierda)
3. Selecciona **Import from File**
4. Elige el archivo `.json` del módulo correspondiente
5. ¡Listo! El flujo aparecerá en tu editor
